# Reminders
Android App to remind to-do tasks using Firebase

## Requirements
* [Android Library](https://github.com/vinkas-technologies/android-library)

## Implementations
* Firebase Database
* Firebase RemoteConfig
* Google Mobile Ads
* Material Design
